# WeatherAPI-rick-and-morty-INSPIRED
An interactive weather app inspired by Rick & Morty, giving you multiverse forecasts for any city (or planet).

# Features:

Search weather by city/planet.

Shows: Temperature, Feels like, Weather description, Humidity, Wind, Pressure.

5-day forecast in scrollable cards.

Dynamic background changes by weather (Sunny, Clouds, Rain, Snow, Thunderstorm).

Loader animation while fetching data.

Press Enter to search quickly.

# How to Use:

Open index.html in a browser.

Enter a city or planet and click "Scan Dimension" (or press Enter).

View current weather and 5-day forecast; background updates automatically.

# Technology:

HTML5

CSS3 (animations & dynamic backgrounds)

JavaScript (Fetch API & DOM manipulation)

OpenWeatherMap API

# How It Works:

User searches for a city.

JS fetches data from OpenWeatherMap.

Current weather displayed in boxes, 5-day forecast in scrollable cards.

Background changes according to main weather condition.

# Customization:

Replace apiKey in script.js with your OpenWeatherMap API key.

Change default city in getWeather("New york").

Edit style.css to adjust colors, fonts, or box styles.
